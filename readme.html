我主要的修改一个是booking-information.php。这个就是直接在主目录下。这个文件就是当用户要book某一个酒店房型的时候，要求用户填写各样的信息。同时也会计算酒店总价并且发送到用户的邮箱。 第二个文件就是在wp-content/themes/twentytwentyfour,这个文件夹下的function.php。当时我上网搜wordpress，很多人说最简单的渲染方式是写short code。然后直接在wordpress里面写[short code]。 但是现在看来也并不是最好的选择。可惜积重难返，功能全部写到了这里面，再想去修改实在太难。但是阅读起来还是比较方便，从第209行往后都是我的代码了。每一个都写明白这个short code是处理哪一个功能的。 基本功能都有实现，但是细节上没有仔细打磨。基本修改的就是这两个文件，还写了一个check-availibility.php，但这是封装了一个简单的功能，几乎不需要修改。因为wordpress确实是边学边写，写的有点乱，还请担待。 哈哈，剩下的还有什么不明白的也可以直接联系我。
